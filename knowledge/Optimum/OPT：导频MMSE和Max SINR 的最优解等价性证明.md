## 1. 核心直觉 (TL;DR) Too Long; Didn't Read

在理想的线性阵列滤波空间中，**最小均方误差 (MMSE) 与 最大信干噪比 (Max SINR) 的最优波束成形权重向量在“空间方向”上是完全一致的，二者仅相差一个标量常数。**

- **Max SINR 视⻆ (功利主义)：** 追求输出功率比值最大，因此必须在干扰方向形成零陷。它不在乎输出波形的绝对幅度。
    
- **MMSE 视⻆ (完美主义)：** 追求输出波形逼近参考信号。为了让波形完美，它**被迫**去做和 Max SINR 一样的事（压制干扰，提取信号）。
    
- **标量常数的意义：** MMSE 最优解比 Max SINR 多出的标量常数，本质上是一个**自动增益控制器 (AGC)**，用于将纯净信号的幅度缩放至与参考波形完全一致，从而实现真正的“均方误差最小”。

## 2. 系统模型与符号定义

假设一个天线阵列接收到的信号向量为 $x$，它由期望信号和“干扰+噪声”组成：

$$x = s \cdot a + n_{in}$$

- $s$: 期望信号波形 (标量)，假设其发射功率为 $P_s = E[|s|^2]$。
    
- $a$: 期望信号的空间导向矢量 (阵列流形，Array Manifold)。
    
- $n_{in}$: 干扰加噪声向量，其协方差矩阵为 $R_{in} = E[n_{in} n_{in}^H]$。
    

接收信号的总协方差矩阵 $R_{xx}$ 可以表示为信号部分与干扰噪声部分的叠加（假设信号与干扰噪声独立）：

$$R_{xx} = E[xx^H] = P_s a a^H + R_{in}$$

## 3. 数学推导过程

### Step 1: Max SINR 最优解回顾

基于广义瑞利商定理，最大化 SINR 的最优权向量为干扰噪声协方差矩阵的逆乘以信号方向：

$$w_{MaxSINR} = R_{in}^{-1} a$$

此时系统能达到的最大 SINR (即广义特征值) 记为 $\rho$：

$$\rho = P_s a^H R_{in}^{-1} a$$

### Step 2: MMSE 最优解回顾

MMSE 准则的目标是最小化输出信号 $w^H x$ 与参考信号 $s$ 之间的均方误差。

根据 Wiener-Hopf 方程，其最优解为：

$$w_{MMSE} = R_{xx}^{-1} r_{xd}$$

其中 $r_{xd}$ 是输入信号 $x$ 与期望信号 $s$ 的互相关向量：

$$r_{xd} = E[x \cdot s^*] = E[(s \cdot a + n_{in}) s^*] = P_s a$$

代入得到 MMSE 的初始解形式：

$$w_{MMSE} = P_s R_{xx}^{-1} a$$

### Step 3: 核心桥梁 —— Woodbury 矩阵求逆引理

观察到 $R_{xx} = R_{in} + P_s a a^H$ 是一个“满秩矩阵 + 秩1矩阵”的形式。为了将 $R_{xx}^{-1}$ 转化为 $R_{in}^{-1}$ 的表达，引入 **Woodbury 矩阵求逆引理**：

$$(A + UCV)^{-1} = A^{-1} - A^{-1}U(C^{-1} + VA^{-1}U)^{-1}VA^{-1}$$

应用于我们的模型：

$$R_{xx}^{-1} = (R_{in} + P_s a a^H)^{-1} = R_{in}^{-1} - \frac{P_s R_{in}^{-1} a a^H R_{in}^{-1}}{1 + P_s a^H R_{in}^{-1} a}$$

### Step 4: 结果代入与化简

我们需要计算 $w_{MMSE}$ 表达式中的核心部分 $R_{xx}^{-1} a$。将上述展开式两边同乘向量 $a$：

$$R_{xx}^{-1} a = R_{in}^{-1} a - \frac{P_s R_{in}^{-1} a (a^H R_{in}^{-1} a)}{1 + P_s a^H R_{in}^{-1} a}$$

注意到分子中的 $P_s a^H R_{in}^{-1} a$ 正是我们前面定义的标量 $\rho$ (最大 SINR)。

进行代换：

$$R_{xx}^{-1} a = R_{in}^{-1} a - \frac{\rho}{1 + \rho} R_{in}^{-1} a$$

提取公因式 $R_{in}^{-1} a$：

$$R_{xx}^{-1} a = \left( 1 - \frac{\rho}{1 + \rho} \right) R_{in}^{-1} a = \frac{1}{1 + \rho} R_{in}^{-1} a$$

### Step 5: 得出最终等价关系

将化简后的 $R_{xx}^{-1} a$ 代回 Step 2 中的 $w_{MMSE}$ 表达式：

$$w_{MMSE} = P_s \left( \frac{1}{1 + \rho} \right) R_{in}^{-1} a$$

$$w_{MMSE} = \left( \frac{P_s}{1 + \rho} \right) w_{MaxSINR}$$

## 4. 结论与工程启发

推导证明，MMSE 的最优波束成形权向量与 Max SINR 的最优权向量严格共线。

**核心启发：**

在解决复杂的抗干扰硬件优化问题（如带有恒模约束/离散相位约束的 RIS 或 DMA 架构）时，直接优化 SINR 往往由于分式形式和约束的存在而成为非凸的 NP-hard 问题。

利用这一等价性，[[工程上将SINR的目标函数替换为无约束的 MMSE 形式]] (去拟合参考波形)**。求解出连续理想空间下的波束指向后，再向离散的硬件码本空间进行投影。因为只要均方误差被压到最小，其空间波束就必然已经对准了最大化 SINR 的最优抗干扰方向。

## 5. 终极闭环：MMSE 的极限性能 ($J_{min}$)

既然 MMSE 和 Max SINR 在方向上是殊途同归的，那么 MMSE 算法收敛后，系统残留的**最小均方误差 (Minimum MSE, $J_{min}$)** 到底是多少？它与系统的极限能力 $\rho$ 之间又存在着极其优美的代数关系。

### Step 1: 写出均方误差 (MSE) 的通用展开式

定义误差 $e = s - w^H x$，均方误差 $J(w)$ 为误差的能量：

$$J(w) = E[|e|^2] = E[(s - w^H x)(s^* - x^H w)]$$

将其彻底展开：

$$J(w) = E[s s^*] - E[s x^H] w - w^H E[x s^*] + w^H E[x x^H] w$$

代入已知定义的协方差与互相关变量：

- 信号总功率：$E[s s^*] = P_s$
    
- 互相关向量：$E[x s^*] = r_{xd}$ (同理 $E[s x^H] = r_{xd}^H$)
    
- 总协方差矩阵：$E[x x^H] = R_{xx}$
    

得到任意权重 $w$ 下的通用误差公式：

$$J(w) = P_s - r_{xd}^H w - w^H r_{xd} + w^H R_{xx} w$$

### Step 2: 代入最优解求 $J_{min}$

将 MMSE 的最优解 $w_{MMSE} = R_{xx}^{-1} r_{xd}$ （及其共轭转置 $w_{MMSE}^H = r_{xd}^H R_{xx}^{-1}$）代入通用公式：

$$J_{min} = P_s - r_{xd}^H (R_{xx}^{-1} r_{xd}) - (r_{xd}^H R_{xx}^{-1}) r_{xd} + (r_{xd}^H R_{xx}^{-1}) R_{xx} (R_{xx}^{-1} r_{xd})$$

观察最后一项，中间的 $R_{xx}^{-1} R_{xx}$ 抵消为单位阵 $I$：

$$J_{min} = P_s - 2 r_{xd}^H R_{xx}^{-1} r_{xd} + r_{xd}^H R_{xx}^{-1} r_{xd}$$

合并后得到 $J_{min}$ 的经典紧凑形式：

$$J_{min} = P_s - r_{xd}^H R_{xx}^{-1} r_{xd}$$

_(直觉：初始总误差是 $P_s$，后面的项就是阵列滤波成功消除掉的误差能量)_

### Step 3: 引入导向矢量与 Woodbury 结论

已知 $r_{xd} = P_s a$，代入减号后面的项：

$$r_{xd}^H R_{xx}^{-1} r_{xd} = (P_s a^H) R_{xx}^{-1} (P_s a) = P_s^2 (a^H R_{xx}^{-1} a)$$

调用我们在第 3 节 (Step 4) 中利用 Woodbury 引理得到的结论：$R_{xx}^{-1} a = \frac{1}{1 + \rho} R_{in}^{-1} a$。

两边同乘 $a^H$ 得到：

$$a^H R_{xx}^{-1} a = \frac{1}{1 + \rho} (a^H R_{in}^{-1} a)$$

### Step 4: 终极坍缩

回忆最大 SINR $\rho$ 的定义式：$\rho = P_s a^H R_{in}^{-1} a$。

移项可得：$a^H R_{in}^{-1} a = \frac{\rho}{P_s}$。

将它代回 Step 3 的末尾：

$$a^H R_{xx}^{-1} a = \frac{1}{1 + \rho} \cdot \frac{\rho}{P_s}$$

最后，把这块拼图放回 Step 2 的 $J_{min}$ 公式中：

$$J_{min} = P_s - P_s^2 \left( \frac{\rho}{P_s(1 + \rho)} \right)$$

约掉分母的 $P_s$，并提取公因式：

$$J_{min} = P_s \left( 1 - \frac{\rho}{1 + \rho} \right) = P_s \left( \frac{1 + \rho - \rho}{1 + \rho} \right)$$

最终得出闭式解：

$$J_{min} = \frac{P_s}{1 + \rho}$$

## 6. 物理意义总结与定性分析

公式 $J_{min} = \frac{P_s}{1 + \rho}$ 揭示了抗干扰系统最底层的物理定律：**系统的残余波形误差，完全由其所能达到的极限 SINR 决定。**

- **理想状态 ($\rho \to \infty$)**：当环境极其干净（无干扰），或天线阵列规模极大、抗干扰能力极强时，$\rho$ 趋于无穷大。此时 $J_{min} \to 0$。这意味着系统输出波形完美拟合了期望波形。
    
- **瘫痪状态 ($\rho \to 0$)**：当压制性干扰极其猛烈，且干扰方向与期望信号方向完全重合，导致系统无法挖出有效的空间零陷时，$\rho$ 趋于 0。此时 $J_{min} \to P_s$。这意味着输出误差的能量等于期望信号本身的能量，系统彻底被干扰淹没。
    

**终极结论：** 无论是朝着“最大化功率比 (SINR)”去优化，还是朝着“最小化波形误差 (MMSE)”去优化，指引算法走向最优解的，始终是同一个底层战力天花板——$\rho$。
