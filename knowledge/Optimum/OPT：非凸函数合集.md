## 一、 判定非凸的标准：两点连线测试

在数学上，一个集合是**凸集**的充要条件是：集合内任意两点的连线（线段）上的所有点，必须仍然在该集合内。

- **凸集**：连接线段在集合“肚子”里。
    
- **非凸集**：连接线段会“穿出”集合，跑到外面去。

## 二、 常见非凸对象分类及成因

### 1. 恒模与球面约束 (Constant Modulus)

这是你在波束赋形中最常遇到的。

- **表达式**：$|w_n| = 1$ 或 $\|\mathbf{w}\|_2^2 = 1$。
    
- **为什么非凸**：它只允许变量待在圆周（或球面）上，而圆心是不被允许的。
    
    - _测试_：取 $A=1, B=-1$，中点 $0$ 不满足 $|0|=1$。
        
- **修改边界能变凸吗？**
    
    - **能。** 将等于改为小于等于：$|w_n| \leq 1$。
        
    - **效果**：集合从“圆周”变成了包含内部的“圆盘”。圆盘内任意两点连线都在圆盘内，变成了**凸集**。
        

### 2. 离散/整数约束 (Discrete/Integer)

- **表达式**：$x \in \{0, 1\}$ 或 $x \in \{1, 2, 3, 4\}$。
    
- **为什么非凸**：集合是由孤立的点组成的，点与点之间是断开的空洞。
    
- **修改边界能变凸吗？**
    
    - **能。** 将其修改为连续区间：$0 \leq x \leq 1$。
        
    - **效果**：这被称为 **Box Constraint（盒约束）**，它是典型的凸集。
        

### 3. 矩阵的秩约束 (Rank Constraint) 

- **表达式**：$\text{rank}(\mathbf{X}) = 1$。
    
- **为什么非凸**：两个秩为 1 的矩阵相加，结果通常是秩为 2 的矩阵。
    
- **修改边界能变凸吗？**
    
    - **不能直接通过等号改不等号解决。** 即使改为 $\text{rank}(\mathbf{X}) \leq 1$ 依然是非凸的。
        
    - **替代方案**：通常使用**核范数（Nuclear Norm）** $\|\mathbf{X}\|_*$ 作为凸替代。

### 4. 乘积项与分式函数 (Bilinear & Rational)

- **表达式**：$f(x, y) = xy$ 或 $f(x) = \frac{1}{x}$（在包含 0 的区间内）。
    
- **为什么非凸**：变量之间的耦合导致曲面出现了“马鞍形”。
    
- **修改边界能变凸吗？**
    
    - **视情况而定。** 对于 $f(x) = 1/x$，如果限制 $x > 0$，它在正半轴是凸的；如果包含负半轴，整体是非凸的。

## 三、 总结：修改边界对凸性的影响表
| **原始非凸约束**                    | **修改后的凸约束**                  | **变化本质 (为什么变凸了)**     | **工程常用术语**                                 |                         |                           |
| ----------------------------- | ---------------------------- | --------------------- | ------------------------------------------ | ----------------------- | ------------------------- |
| **$                           | w                            | = 1$** (圆周/球面)        | **$                                        |                         |                           |
| **$x \in \{0, 1\}$** (二进制)    | **$0 \leq x \leq 1$** (连续区间) | 从“孤立的两个点”变为“连续的线段”。   | **线性/连续松弛 (Linear/Continuous Relaxation)** |                         |                           |
| **$\\                         | x\\                          | _2^2 \geq$** (球外区域)   | **无** (若改 $\leq$ 意义则反转)                    | 原本排除了原点（中空），若坚持向外则永远非凸。 | **逆凸约束 (Reverse Convex)** |
| **$\text{rank}(X) = 1$** (低秩) | __$\|X\|__ \leq C$_* (核范数)   | 将离散的“个数”统计变为连续的“能量”和。 | **半正定松弛 (SDR)**                            |                         |                           |
> **核心规律**：
> 
> 1. **“等于号”通常是非凸的罪魁祸首**。在非线性空间中，等式约束往往对应着一个流形的边缘。
>     
> 2. **向内填充通常能变凸**。如果把一个“环”填充成“饼”，把“点”拉成“线”，通常就能获得凸性。
>     
> 3. **向外扩展通常无效**。例如要求变量在圆圈外面（$|w| \geq 1$），这依然是一个中间有空洞的集合，是非凸的。
>

## 四、 为什么我们不直接修改边界？

虽然把 $|w|=1$ 改成 $|w| \leq 1$ 之后问题变简单了，但**物理意义变了**：

- 在波束赋形中，如果天线功率不够（即 $|w| < 1$），发射出的信号强度会打折扣，可能无法达到硬件的最大效能。
    
- 因此，我们通常先在修改后的凸边界上求一个最优解，再通过**归一化（Normalization）**等手段强行拉回原来的非凸边界。这就是为什么需要 ADMM 这种既能利用凸优化便利、又能兼顾非凸硬约束的算法。
