## 1. 明确优化目标与约束 (LCMV 准则)

Frost 算法的本质是求解 **线性约束最小方差 (LCMV)** 问题。

假设天线阵元数为 $N$，约束条件有 $K$ 个。

- **目标函数 (极简防守)：** 最小化阵列输出的总功率。
    
    在自适应迭代中，我们无法知道统计协方差矩阵 $R_{xx}$，所以用**瞬时输出功率**代替期望功率：
    
    $$J(w) = \frac{1}{2} |y(n)|^2 = \frac{1}{2} w^H x(n) x^H(n) w$$
    
    _(注：乘以 $\frac{1}{2}$ 是为了后续求导时刚好约掉系数，不影响极值点)_
    
- **约束条件 (护住头部)：**
    
    $$C^H w = f$$
    
    - $C$ 是一个 $N \times K$ 的约束矩阵（例如，某一列就是有用信号的导向矢量 $a$）。
        
    - $f$ 是一个 $K \times 1$ 的目标响应向量（例如 $f = 1$）。

## 2. 第一步：无约束梯度下降 (盲目下山)

如果我们暂时不管约束条件，只为了让瞬时输出功率 $J(w)$ 变小，我们需要计算梯度。

对目标函数 $J(w)$ 针对共轭权重 $w^*$ 求导（复矢量求导法则）：

$$\nabla_w J(n) = \frac{\partial}{\partial w^*} \left[ \frac{1}{2} w^H x(n) x^H(n) w \right] = x(n) x^H(n) w = x(n) y^*(n)$$

利用最速下降法，我们顺着梯度的反方向走一步，得到一个**临时权重** $w_{temp}$：

$$w_{temp}(n+1) = w(n) - \mu \nabla_w J(n)$$

$$w_{temp}(n+1) = w(n) - \mu x(n) y^*(n)$$

_($\mu$ 为迭代步长。这一步就是我们之前说的不管三七二十一，顺着梯度把所有声音调小。)_

---

## 3. 第二步：拉格朗日法与几何投影 (强行拉回)

刚才算出的 $w_{temp}(n+1)$ 百分之百破坏了 $C^H w = f$ 的约束。我们需要在这个临时权重的基础上，加上一个**修正量** $\Delta w$，把它重新拉回到约束平面上。

设最终的合法权重为：

$$w(n+1) = w_{temp}(n+1) + \Delta w$$

**核心几何逻辑：** 为了让这种拉回对刚才梯度下降的成果破坏最小（即移动距离最短），修正量 $\Delta w$ 必须垂直于约束平面。在数学上，这意味着 $\Delta w$ 必须落在约束矩阵 $C$ 的列空间中。

因此，我们可以设 $\Delta w = C \lambda$（其中 $\lambda$ 是待求的拉格朗日乘子向量）。

代入上式得到：

$$w(n+1) = w_{temp}(n+1) + C \lambda$$

现在，我们强令这个最终权重 $w(n+1)$ 必须满足约束条件 $C^H w = f$。等式两边同乘 $C^H$：

$$C^H w(n+1) = C^H w_{temp}(n+1) + C^H C \lambda = f$$

这是一个关于 $\lambda$ 的线性方程，解出 $\lambda$：

$$C^H C \lambda = f - C^H w_{temp}(n+1)$$

$$\lambda = (C^H C)^{-1} \left[ f - C^H w_{temp}(n+1) \right]$$

将求出的 $\lambda$ 重新代回到 $w(n+1)$ 的表达式中：

$$w(n+1) = w_{temp}(n+1) + C (C^H C)^{-1} \left[ f - C^H w_{temp}(n+1) \right]$$

将式子展开并重新合并同类项 $w_{temp}(n+1)$：

$$w(n+1) = w_{temp}(n+1) - C (C^H C)^{-1} C^H w_{temp}(n+1) + C (C^H C)^{-1} f$$

$$w(n+1) = \left[ I - C (C^H C)^{-1} C^H \right] w_{temp}(n+1) + C (C^H C)^{-1} f$$

---

## 4. 终极组装：Frost 算法标准更新公式

为了让公式在工程实现中极致简洁，Frost 预先定义了两个在迭代过程中**永远不变**的常量矩阵和向量。

**1. 投影矩阵 $P$ (Projection Matrix):**

$$P = I - C (C^H C)^{-1} C^H$$

_(它的物理意义：将任意向量投影到正交于约束矩阵 $C$ 的子空间中)_

**2. 辅助向量 $F$ (Constrained Vector):**

$$F = C (C^H C)^{-1} f$$

_(它的物理意义：在满足约束条件的所有向量中，模长最短的那个基础向量)_

将 $P$ 和 $F$ 代入上一节的末尾式，并把第一步推导的 $w_{temp}(n+1)$ 完整代入：

$$w(n+1) = P \left[ w(n) - \mu x(n) y^*(n) \right] + F$$

**这就是 Frost 算法名震江湖的最终迭代公式。**

## 5. 补充：算法的初始化

由于迭代公式中的 $P$ 操作会抹掉任何不符合约束的分量，为了让算法赢在起跑线上，初始权重 $w(0)$ 必须严格满足约束条件。

通常最安全的初始化方法，就是直接使用辅助向量 $F$：

$$w(0) = F = C (C^H C)^{-1} f$$

_(注：如果约束只是简单的单个方向 $a$，且 $f=1$，那么 $w(0) = \frac{a}{a^H a}$，也就是我们常说的常规波束成形 (CBF) 或静态波束权重。)_
