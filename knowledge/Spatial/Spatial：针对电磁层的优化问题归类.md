## 1. 重构阵列的空间拓扑结构
分类依据：$F = \mathbf{w}^H (\mathbf{EF} \odot \mathbf{v})$。

| **理论分类阵营**                                           | **天线技术名称**                                              | **数学本质                                                                                         | **工作机制**                                        | **阵列模型中的体现**                                                                                            |
| ---------------------------------------------------- | ------------------------------------------------------- | ---------------------------------------------------------------------------------------------- | ----------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| 空间位置重构<br><br>(改变 $\mathbf{v}$)                      | 传统可移动天线<br><br>(3D MA)                                  | **优化空间相位向量** $\mathbf{v}(\mathbf{p})$<br><br>  <br><br>获取位置自由度                                 | 机械驱动固态天线阵元在连续空间内平移。                             | $EF$ 视为常量（或理想全向），通过寻优坐标 $\mathbf{p} = [x, y, z]^T$，利用空间衰落的剧烈变化来实现相位对齐或干扰避让。                             |
|                                                      | 流体天线系统<br>(FAS)                                         | **优化空间相位向量** $\mathbf{v}(\mathbf{p})$<br><br>  <br><br>获取“位置/端口自由度”                            | 驱动液态金属在介质管道中流动，改变有效导电区域的位置。                     | 与 MA 高度类似，只是位置 $\mathbf{p}$ 的轨迹受限于管道拓扑结构。                                                               |
|                                                      | 像素天线 /<br><br>  <br><br>辐射中心重构<br><br>  <br><br>(RCRAA) | **离散化优化空间相位** $\mathbf{v}(\mathbf{r})$<br><br>  <br><br>获取“网格选择自由度”                            | 通过 PIN 阵列开关，将复杂的微带表面选通/选断，使得天线的“有效相位中心”在表面发生跳跃。 | 将连续的坐标寻优降维成从高密度网格中提取子集的组合优化问题（布尔选择向量 $\mathbf{r}$）。                                                     |
| 电磁辐射重构<br><br>  <br><br>(改变 $EF$)                    | 动态超表面天线<br><br>  <br><br>(DMA)                          | **优化单元辐射包络** $\mathbf{EF}(\mathbf{\Phi})$<br><br>  <br><br>获取“电磁响应自由度”                         | 不改变物理坐标，通过改变贴片上的二极管偏置电压，调节洛伦兹谐振器的局部透射/反射系数。     | 空间相位 $\mathbf{v}$ 固定死板，但 $\mathbf{EF}$ 变成了一个由电压矩阵 $\mathbf{\Phi}$ 控制的可调向量，能实现原地的幅度、相位和极化重构。             |
|                                                      | 可重构智能表面<br><br>  <br><br>(RIS / IRS)                    | **优化等效辐射包络** $\mathbf{EF}(\mathbf{\Theta})$<br><br>  <br><br>获取传播环境自由度                         | 不改变物理坐标，通过调节反射单元的阻抗，改变入射电磁波的相位/幅度。              | 若作为接收机的一部分，其本质是将原本杂乱的电磁波在固定位置进行相干叠加，等效于赋予了该位置一个极度灵活的 $EF$ 接收包络。                                         |
| 跨界阵营：<br><br>全维度重构<br><br>(同时改变 $\mathbf{v}$ 和 $EF$) | 六维可移动天线<br><br>  <br><br>(6DMA)                         | **联合优化 $\mathbf{EF}(\mathbf{A}) \odot \mathbf{v}(\mathbf{p})$**<br><br>  <br><br>获取位置+姿态双重自由度” | 既包含机械平移（改变位置），又包含机械旋转（改变俯仰角和方位角姿态）。             | 平移改变 $\mathbf{v}(\mathbf{p})$ 避开深衰落；旋转改变 $\mathbf{EF}(\mathbf{A})$ 使得单天线的物理零陷对准干扰源、或者极化正交。是目前自由度最彻底的模型。 |
