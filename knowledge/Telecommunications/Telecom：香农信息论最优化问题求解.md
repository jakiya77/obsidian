
### 1. 明确目标：我们要最大化什么？

在连续域中，假设信号的概率密度函数为 $f(x)$，它的**微分熵（信息熵）**定义为：

$$h(f) = -\int_{-\infty}^{+\infty} f(x) \ln f(x) dx$$

我们的目标是：找到一个特定的函数形状 $f(x)$，使得这个积分的结果（信息熵）达到最大值。

### 2. 物理现实的约束条件

我们不能毫无限制地放大信号。在实际通信中，存在以下几个必须满足的物理和数学约束：

- **约束1（概率守恒）**：概率密度函数的总面积必须为 1。
    
    $$\int_{-\infty}^{+\infty} f(x) dx = 1$$
    
- **约束2（均值为0）**：为了方便，我们假设信号没有直流偏置，均值为 0（即便均值不为0，也只会平移波形，不影响分布的形状）。
    
    $$\int_{-\infty}^{+\infty} x f(x) dx = 0$$
    
- **约束3（功率受限）**：信号的平均功率（即方差 $\sigma^2$）是固定有限的。
    
    $$\int_{-\infty}^{+\infty} x^2 f(x) dx = \sigma^2$$
    

### 3. 请出数学工具：变分法 + 拉格朗日乘子法

在普通微积分中，如果你要在满足 $g(x)=0$ 的条件下去求 $f(x)$ 的极值，你会引入拉格朗日乘子 $\lambda$，构造一个新函数 $L(x) = f(x) + \lambda g(x)$，然后求导令其为 $0$。

**变分法（Calculus of Variations）**的逻辑完全一样，只不过未知数从“变量 $x$”变成了“函数 $f(x)$”。 我们将目标函数（信息熵）和三个约束条件结合起来，引入三个拉格朗日乘子 $\lambda_0, \lambda_1, \lambda_2$，构造出一个**泛函（Functional）**的被积函数 $L(f)$：

$$L(f) = -f(x) \ln f(x) + \lambda_0 f(x) + \lambda_1 x f(x) + \lambda_2 x^2 f(x)$$

根据变分法中的**欧拉-拉格朗日方程（Euler-Lagrange Equation）**，要让泛函取得极值，被积函数 $L$ 对函数 $f$ 的偏导数必须为 0。

### 4. 见证奇迹的求导过程

我们把 $f(x)$ 就当成一个普通的自变量，对 $L(f)$ 求偏导：

$$\frac{\partial L}{\partial f} = -\left( \ln f(x) + f(x) \cdot \frac{1}{f(x)} \right) + \lambda_0 + \lambda_1 x + \lambda_2 x^2 = 0$$

化简一下括号里的项：

$$- (\ln f(x) + 1) + \lambda_0 + \lambda_1 x + \lambda_2 x^2 = 0$$

现在，我们把 $\ln f(x)$ 留在等式左边，其余全移到右边：

$$\ln f(x) = (\lambda_0 - 1) + \lambda_1 x + \lambda_2 x^2$$

为了把 $f(x)$ 解出来，我们对两边同时取自然指数 $e$：

$$f(x) = e^{(\lambda_0 - 1) + \lambda_1 x + \lambda_2 x^2}$$

你看！**没有任何人为的预设，仅仅是求了一个导数，指数函数的形态 $e^{ax^2+bx+c}$ 就自然而然地跃然纸上了！**

### 5. 确定常数，高斯分布现身

现在，我们只需要用最初始的三个约束条件，把未知常数 $\lambda_0, \lambda_1, \lambda_2$ 给反解出来：

1. **利用“概率守恒”和“方差有限”**：既然 $f(x)$ 是个概率密度，它在正负无穷大的地方必须趋近于 0（否则积分会爆炸）。这就要求指数最高次项 $x^2$ 的系数必须是负数，即 $\lambda_2 < 0$。
    
2. **利用“均值为0”**：函数必须是关于 $y$ 轴对称的偶函数，所以一次项 $x$ 的系数必须为 0，即 $\lambda_1 = 0$。
    
3. 我们将 $f(x) = e^{(\lambda_0 - 1)} e^{\lambda_2 x^2}$ 代入积分式 $\int f(x) dx = 1$ 和 $\int x^2 f(x) dx = \sigma^2$ 中，通过解高斯积分（这里省去繁琐的代数运算），会得出：
    
    $$e^{(\lambda_0 - 1)} = \frac{1}{\sqrt{2\pi\sigma^2}}$$
    
    $$\lambda_2 = -\frac{1}{2\sigma^2}$$
    

把它们代回 $f(x)$ 的表达式，最终的形态就是：

$$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{x^2}{2\sigma^2}}$$

这就是完美的高斯分布公式！
