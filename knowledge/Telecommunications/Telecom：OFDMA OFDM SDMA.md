# 1) 基本概念（

- **OFDM（正交频分复用）**：单用户调制方式，把一个宽带拆成很多正交子载波，解决频率选择性衰落；**不直接负责“多用户资源分配”**。
    
- **OFDMA（正交频分多址）**：在 OFDM 之上做**多用户接入**——把不同**子载波/资源块**分配给不同用户，实现**频域/子载波维度**的多址。
    
- **SDMA（空分多址）**：利用**空间维度（波束/阵列）**来同时服务多用户——同一时频资源上，用不同的**空间波束**把用户“隔开”。
    

# 2) 资源维度的核心区别

- **OFDMA**：靠“**频率**”把用户分开 → 不同用户占不同子载波（或PRB）；同一时隙内互不干扰（理想调度下）。
    
- **SDMA**：靠“**空间**”把用户分开 → 不同用户共享**相同的时频**，但通过波束赋形/预编码让彼此干扰尽量为零（或够小）。
    

# 3) 体系差异与工程含义

**(A) 干扰处理与编解码复杂度**

- **OFDMA**：天然“正交”分频，多用户间基本不互相干扰（跨用户 ICI 小）；调度就是“给谁哪些子载波”。
    
- **SDMA**：共享同频同时间，必须靠**多天线预编码/波束形成**抑制**多用户干扰（MUI）**；需要较强的 CSI 与矩阵运算（如 ZF/MMSE）。
    

**(B) 硬件/射频链路约束**

- **数字波束成形（DBF）**：每根天线后都有独立 RF/ADC/DAC，可做**多流预编码**，天然适合 **SDMA（多用户同时同频）**。
    
- **模拟波束成形（ABF）**：一条/少数 RF 链 + 相移网络，多流并发困难（相位/幅度受限、等模约束）；更适合把**不同用户分到不同子载波** → **OFDMA**。
    

**(C) 资源利用率与调度自由度**

- **OFDMA**：对频率选择性友好，能按用户的**频域信道优势**做“子载波匹配”；但同一子载波上**很难并发多用户**。
    
- **SDMA**：时频资源复用度高（同频并发），但要求用户之间在**角度/位置/通道子空间**上“够可分”。
    

**(D) 信道状态信息（CSI）需求**

- **OFDMA**：需要频域 CSI 做子载波调度；
    
- **SDMA**：更依赖**高维空间 CSI**（阵列域/近场通道），用于构造低互相关的多用户波束（如 ZF 需要 (H) 的伪逆）。
    

# 4) 与“可移动天线（MA）+ 近场”模型的关系（呼应论文摘要）

- **近场效应**：通道不再是纯“角度扇区”分离，而是**距离-角度**联合分辨；MA 可通过**移动阵元/子阵**改变**阵面几何与相位中心**，增强用户间的**空间可分性**。
    
- **数字架构 + SDMA（文中做法）**：
    
    - 用 **ZF** 等数字预编码，联合优化 **APV（阵元/子阵位置向量）+ 预编码矩阵**，让用户通道**近似正交**，逼近期望的“最小用户速率上界”。
        
    - MA 的自由度帮助把用户“拉开”，从而让 ZF 更干净、MUI 更小。
        
- **模拟架构 + OFDMA（文中做法）**：
    
    - ABF 难以同时发多流，因此**把多用户“搬到”不同子载波**；每个子载波上只需形成**单用户最优模拟波束**（ABFV），再联合**功率分配 + APV**。
        
    - MA 仍能降低波束间串扰、减少运动频率（统计信道设计，APV 低频更新）。
        

# 5) 什么时候用谁？

- **用户数不多、基站有足够 RF 链 & 想最大化同频复用** → **数字 + SDMA**：
    
    - 条件：CSI 可靠、用户在空间上可分（近场/MA 可帮忙提高可分性）。
        
    - 目标：提高系统吞吐与频谱效率。
        
- **RF 链受限（功耗/成本）、或想降低多用户预编码复杂度** → **模拟 + OFDMA**：
    
    - 条件：频域可调度、能做子载波级资源分配与功控。
        
    - 目标：在硬件受限下稳妥实现多用户服务。
        
- **混合波束成形**场景：常见工程折中是 **“OFDMA ×（每RB内做小规模SDMA）”**——先按频域分配，再在每个资源块上做少量空间复用；MA/近场有助于把“可同时上的用户”挑出来。

# 7) 一句话总结 

- **OFDM** 是“单用户多载波调制”；
    
- **OFDMA** 用“**频率**”把多用户**分开**；
    
- **SDMA** 用“**空间**/波束”让多用户**共用同一时频**；
    
- **数字架构→SDMA** 合理，**模拟架构→OFDMA** 合理；
    
- **MA + 近场** 提升“空间可分性”，让 **SDMA 更干净、OFDMA 更稳健**。
