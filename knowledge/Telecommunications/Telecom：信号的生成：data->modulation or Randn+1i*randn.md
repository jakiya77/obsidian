### 一、 核心差异速查表

| **对比维度**    | **x = randn + 1i*randn (复高斯随机信号)**               | **data = randi([0 1]) (离散二进制比特流)**        |
| ----------- | ------------------------------------------------ | ----------------------------------------- |
| **物理本质**    | 模拟具备理想统计特性的电磁波/白噪声                               | 模拟实际要传输的数字化信息（0和1）                        |
| **仿真层级**    | 阵列信号处理级、物理层理论极限                                  | 完整的端到端数字通信链路                              |
| **核心关注点**   | 信号的**空间特征**、**协方差矩阵**、**功率谱**                    | 信号的**抗干扰恢复能力**、**误码率 (BER)**              |
| **适用算法/场景** | 波达方向估计 (MUSIC, ESPRIT)、波束成形 (MVDR) 理论性能测试、信道容量计算 | 调制解调测试、信道编码 (LDPC/Polar) 验证、整体抗干扰通信系统性能评估 |
| **是否需要调制**  | 否（直接作为发射端的基带复信号）                                 | 是（必须通过 BPSK/QPSK 等映射为复数符号）                |

### 二、 什么时候用 `randn + 1i*randn`？

**核心原则：当你的研究重点是“空间域算法”或“理论性能边界”，而不关心具体传输了什么内容时。**

- **测试阵列信号处理算法（如 MUSIC）：** MUSIC 算法的核心是协方差矩阵 $R_x$ 的特征值分解。复高斯白噪声的二阶统计特性非常完美，能确保不同信号之间是严格不相关的（除非你人为构造多径），从而让算法的性能完全取决于空间角度和信噪比，排除调制符号带来的离散概率波动。
    
- **计算香农信道容量 (Channel Capacity)：** 根据信息论，当发射信号服从高斯分布时，信道容量能够达到理论最大值。因此在评估系统极限潜力时，默认使用高斯信号。[[Telecom：香农信息论（Shannon Information Theory）]]

### 三、 什么时候用 `data = randi([0 1])` 并且如何调制？

**核心原则：当你的最终评估指标包含“误码率 (BER)”，或者你需要验证接收端能否在干扰下把原始信息完整还原出来时。**

如果你生成了 0 和 1 的比特流，它们不能直接乘以信道矩阵，必须经过**数字调制（Digital Modulation）**，将离散的比特映射为星座图上的复数符号（基带信号）。

以下是两种最常用调制方式的公式推导和 MATLAB 代码实现：

#### 1. BPSK 调制 (Binary Phase Shift Keying)

BPSK 是最基础的调制方式，每个符号携带 1 个比特，相位只有两种状态（通常是 $0$ 和 $\pi$）。

- **数学映射规则：**
    
    - 比特 `0` $\rightarrow$ 符号 $+1$
        
    - 比特 `1` $\rightarrow$ 符号 $-1$
        
- **转换公式：** 设比特流为 $b \in \{0, 1\}$，调制后的符号为 $s$，则：
    
    $$s = 1 - 2b$$
    
- **MATLAB 代码：**
    ``` Matlab
    K = 1000; % 假设要发送 1000 个符号
    
    % 1. 生成比特流
    data_bits = randi([0 1], K, 1); 
    
    % 2. BPSK 调制映射
    symbols_bpsk = 1 - 2 * data_bits; 
    
    % 3. 赋予发射功率 P_s (此时信号是实数)
    x_signal = sqrt(P_s) * symbols_bpsk; 
    ```
    

#### 2. QPSK 调制 (Quadrature Phase Shift Keying)

QPSK 利用了复平面的同相（I）和正交（Q）两个维度，每个符号携带 2 个比特，因此频谱效率是 BPSK 的两倍。

- **数学映射规则：**
    
    将比特流两两分组，第一位控制实部（I路），第二位控制虚部（Q路）。
    
    为了保证调制后的信号平均功率为 1（功率归一化），必须除以 $\sqrt{2}$。
    
- **转换公式：** 设比特组为 $(b_I, b_Q)$，则：
    
    $$s = \frac{1}{\sqrt{2}} \left[ (1 - 2b_I) + j(1 - 2b_Q) \right]$$
    
- **MATLAB 代码：**
    
    Matlab
    
    ```
    K = 1000; % 假设要发送 1000 个符号
    
    % 1. 生成比特流 (QPSK需要 2*K 个比特)
    data_bits = randi([0 1], 2*K, 1); 
    
    % 2. 串并转换：拆分 I 路和 Q 路
    data_I = data_bits(1:2:end); % 取奇数位
    data_Q = data_bits(2:2:end); % 取偶数位
    
    % 3. QPSK 调制映射 (带功率归一化)
    symbols_qpsk = ((1 - 2*data_I) + 1i * (1 - 2*data_Q)) / sqrt(2);
    
    % 4. 赋予发射功率 P_s
    x_signal = sqrt(P_s) * symbols_qpsk; 
    ```

无论是生成信号（Signal/Noise）还是生成信道（Channel Fading），只要用到 `randn + 1i*randn`，**永远记得除以 $\sqrt{2}$**。

- 信号除以 $\sqrt{2}$：是为了让发射功率严格等于你设定的 $P$。
    
- 信道除以 $\sqrt{2}$：是为了让小尺度衰落的平均增益保持中立（为 $1$），确保所有的能量衰减都由明确的距离参数（大尺度损耗）来单独掌管。
#### 为什么信道的随机生成也必须遵守能量为 1 的法则：
---
##### 1. $\alpha$ 的物理身份：瑞利衰落（Rayleigh Fading）的源头

在多径环境中，电磁波经过建筑物反射、散射后到达接收端。每一条路径 $l$ 都会对信号造成两件事：

1. **幅度衰减**：能量变弱了。
    
2. **相位偏移**：因为传播距离不同，波的相位发生了旋转。
    

数学上，这就用一个复数 $\alpha = x + jy$ 来表示。

当你用 `(randn + 1i*randn)/sqrt(2)` 时，实部和虚部都是方差为 $0.5$ 的独立高斯分布。根据概率论：

- 这个复数的**模长 $|\alpha|$** 会严格服从**瑞利分布（Rayleigh Distribution）**，完美模拟了幅度随时间的剧烈随机波动。
    
- 这个复数的**相位 $\angle\alpha$** 会严格服从 $[-\pi, \pi]$ 的**均匀分布**，完美模拟了到达相位的完全不可预测性。
    

##### 2. 核心原因：分离“大尺度衰落”与“小尺度衰落”

在严谨的通信链路仿真中，我们将信道对信号能量的消耗严格划分为两部分，绝不允许它们互相串位：

- **大尺度衰落（Large-Scale Fading）：** 这是由**传播距离**（路径损耗）和**大型障碍物遮挡**（阴影衰落）决定的。它决定了信号到达接收机时的**平均功率**究竟是多少。
    
- **小尺度衰落（Small-Scale Fading）：** 就是这里的 $\alpha$。它是由于微小的位置移动或环境变化导致的**局部剧烈波动**（比如多径干涉引起的深衰落）。
    

**归一化的意义就在于此：**

我们希望 $\alpha$ **只负责模拟“波动的形状（分布）”，绝不干涉“平均能量的大小”**。

通过除以 $\sqrt{2}$，我们强制让小尺度衰落系数 $\alpha$ 的数学期望（平均功率增益）等于 1：

$$E[|\alpha|^2] = E\left[\left| \frac{randn + j \cdot randn}{\sqrt{2}} \right|^2\right] = 1$$

##### 3. 如果信道不归一化，会引发什么灾难？

回顾你之前贴出的代码，里面有这样一个非常规范的操作：

`x_signal = sqrt(P_signal_li) * ...`

这里，你已经通过 `P_signal_li` 极其精确地定义了到达接收端的信号功率（这通常是经过大尺度路径损耗计算后得到的值）。

如果你在生成信道 $\alpha$ 时忘了除以 $\sqrt{2}$，那么 $E[|\alpha|^2]$ 就会等于 $2$。

当你的信号穿过这个信道（$y = \alpha \cdot x$）时，**信道会凭空给你的信号放大一倍的能量（平白无故多出 $3\text{ dB}$ 的增益）**！
