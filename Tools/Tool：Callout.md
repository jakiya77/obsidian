| **视觉颜色** | **推荐类型 (Type)**                       | **适用场景**        |
| -------- | ------------------------------------- | --------------- |
| **蓝色**   | note, info                            | 普通笔记、补充信息       |
| **青色**   | abstract, summary, tldr               | 摘要、简述、长话短说      |
| **绿色**   | success, check, done                  | 正确、完成的任务、成功结论   |
| **青色**   | tip, hint, important                  | 小技巧、重要提示、点子     |
| **紫色**   | question, help, faq                   | 疑问、帮助说明、常见问题    |
| **橙色**   | ç, caution, attention                 | 警告、注意事项、需要留意的地方 |
| **红色**   | failure, fail, missing, danger, error | 错误、缺失、严重危险      |
| **深青色**  | bug                                   | 记录程序错误或逻辑漏洞     |
| **灰色**   | example, quote, cite                  | 示例、引用内容         |
|          |                                       |                 |
除了改变类型，你还可以通过语法调整它的**展示形态**：

1. **标准形式**：默认展开，不可收起。 `> [!info] 标题`
    
2. **默认展开（可折叠）**：标题右侧会有一个小箭头，点击可以收起。 `> [!info]+ 标题`
    
3. **默认收起（可折叠）**：进入页面时只显示标题，点击箭头才展开内容。 `> [!info]- 标题`
