
### 💡 2分钟速成：算法到代码的“套娃”逻辑

无论论文里的数学公式写得多吓人（比如带各种 $\max, \arg\max, \sum$），落实到 MATLAB 代码里，永远逃不出这四个核心步骤的俄罗斯套娃：

**1. 初始化 (Initialize) —— “打底稿”**

- **数学含义：** 算法需要一个起点。
    
- **你的代码：** `Phi_curr_cd = zeros(N, 1);` 和 `best_sinr_cd = -inf;`。你让所有天线先指向 0 度，把历史最好成绩设为负无穷大。
    

**2. 维度解耦 (Nested Loops) —— “剥洋葱”**

- **数学含义：** 把一个多变量的极其复杂的函数，拆成一个个单变量函数。
    
- **你的代码（三层套娃）：**
    
    - **第一层（宏观控制）：** `for iter = 1:8` （外层管收敛，跑几圈）。
        
    - **第二层（空间维度）：** `for n = 1:N` （每次只看第 $n$ 根天线）。
        
    - **第三层（物理字典）：** `for k = 1:K` （让第 $n$ 根天线挨个尝试 $K$ 个角度）。
        
    - _(你看，I-PSR 算法其实就是把第二层改成了 `for n = start_idx : 2 : N-1`，把第三层变成了 `for k1... for k2...`，逻辑完全没变！)_
        

**3. 核心评估 (Evaluate) —— “算分数”**

- **数学含义：** 计算目标函数 $\gamma(\mathbf{w}, \mathbf{\Phi})$。
    
- **你的代码：** 就是在最内层那个极其冗长的循环里，算 `v_T_temp`，算 `R_J_temp`，最后祭出神级公式 `current_sinr = real(v_T_temp' * ((R_J_temp + P_N*eye(N)) \ v_T_temp));`。这里是整个代码最耗算力的地方，也就是“搬砖”的地方。
    

**4. 贪心更新 (Update) —— “优胜劣汰”**

- **数学含义：** $\arg\max$ 操作。
    
- **你的代码：**
    
    Matlab
    
    ```
    if current_sinr > best_sinr_n
        best_sinr_n = current_sinr; % 刷新最高分
        best_mode_n = Phi_temp(n);  % 记录下这个带来最高分的“姿势”
    end
    ```
    
    就这一个 `if` 语句，完成了整篇论文里最高大上的 $\arg\max$ 寻优操作！
